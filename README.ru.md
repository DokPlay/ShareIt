# ShareIt

[English version](README.md)

Сервис на Spring Boot для шаринга вещей между пользователями. Предоставляет REST-эндпоинты для управления пользователями и вещами с использованием in-memory хранилища.

## Возможности
- CRUD для пользователей
- CRUD для вещей с проверкой владельца
- Поиск по доступным вещам по тексту
- Централизованная обработка ошибок с понятными HTTP-ответами

## Технологии
- Java 17+
- Spring Boot
- Maven
- Lombok

## Быстрый старт
1. Сборка: `./mvnw clean package`
2. Запуск: `./mvnw spring-boot:run`
3. Порт по умолчанию: 8080

## Тестирование
- Запуск тестов: `./mvnw test`

## Структура проекта
- `src/main/java` — основной код
- `src/test/java` — автотесты
- `src/main/resources` — конфигурация
